# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it responsibly.

**Do not open a public GitHub issue.**

Instead, email **shaun.nichols@conductorone.com** with:

- A description of the vulnerability
- Steps to reproduce
- Potential impact

You should receive a response within 48 hours.

## Scope

This tool runs locally and connects to the ConductorOne API using credentials you provide. It does not store credentials to disk, transmit data to third parties, or listen on network interfaces beyond localhost.

If you discover a vulnerability in the ConductorOne platform itself (not this tool), please report it through ConductorOne's official security channels.
